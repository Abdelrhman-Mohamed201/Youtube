# Youtube-App

> Angular project

## Build Setup

``` bash
# install dependencies
npm run prestart
npm run postinstall

# serve at localhost:8080
npm run start

# e2e-tests
npm run protractor

# compile sass files
npm run sass
```

## Built With
* [angular](https://github.com/angular/angular.js): 1.6.9
* [angular-ui-router](https://ui-router.github.io/ng1/): 1.0.7
* [protractor](https://github.com/angular/protractor): 4.0.9
